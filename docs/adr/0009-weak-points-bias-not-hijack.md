# Weak points bias the prescriber, they don't hijack it

Active weak-point tags are passed into prescription logic as bias signals and
explanation constraints, not as absolute command authority. A limitation is useful
information; it should influence exercise choice, assistance work, and emphasis
*after* safety, goal, and block context are respected.

**Guardrail:** weak points adjust emphasis; they never override safety, goal, or block
context.
