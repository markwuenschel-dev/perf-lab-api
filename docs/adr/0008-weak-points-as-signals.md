# Store weak points as probabilistic signals

Weak points are rows with `tag`, `source`, `confidence`, `note`, and `resolved`
status — not a single static athlete attribute. Weakness can come from self-report,
benchmark, inference, or performance data, and those sources carry different
confidence levels that should not be collapsed into one permanent trait.

**Guardrail:** do not treat weak points as one static athlete attribute.
