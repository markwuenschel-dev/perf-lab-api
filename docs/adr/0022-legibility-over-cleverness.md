# Favor legibility over premature cleverness

Prefer explicit state, dose, plan, benchmark, weak-point, and explanation structures,
accepting some verbosity, because a system that cannot be explained cannot be debugged
or trusted. This is the meta-decision behind most of the others in this log.

**Guardrail:** when in doubt, choose the option that keeps the control loop easy to
inspect and defend.
