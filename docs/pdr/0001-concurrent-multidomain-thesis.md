# Perf Lab is a concurrent, multi-domain adaptive engine

Performance Lab is a **concurrent / hybrid multi-domain adaptive training engine** —
one unified body state, all capacities at once, with cross-talk. The first-class user
is the athlete who runs *and* lifts (Hyrox / CrossFit / tactical / runner-who-lifts),
**multi-sport now**, not running-first. This is the shape the backend already has; the
running-only UI framing is what must broaden.

We considered keeping a running-first product and bolting on other modalities later;
rejected because the engine is already general and the target persona trains
concurrently. (Direction settled 2026-06-19.)

**Guardrail:** new product surfaces must assume a concurrent multi-domain athlete — do
not narrow the framing back to a single sport.
