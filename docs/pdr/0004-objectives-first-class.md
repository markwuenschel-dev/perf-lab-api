# Objectives are a first-class model

What an athlete trains *toward* is a first-class concept — an **Objective**: a target
benchmark + value + date + priority, with many held concurrently. This generalizes the
orphaned Goal/Race screen and feeds the prescriber's stress allocation and taper
windows. Objectives are product-level intent, distinct from `MesocycleBlock` (strategic
structure) and from the prescriber's daily content.

**Guardrail:** model objectives as their own first-class entity (multi-target,
cross-domain) — do not collapse them into a single "goal" field or into blocks.
